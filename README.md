## My dotfiles 
---

### Support Dist
* OpenSuse
* Debian/Ubuntu
* MacOS

### Included Configs
* Bash
* Neovim and vim(may not work)
* Git
* Dev Languages and Tools: py, ruby, go, nodejs

### How to use
``` bash
# get git first
git clone https://github.com/haocs/dotfiles.git "$HOME/.dotfiles"
cd dotfiles/bin

# For deb
source ./setup_deb.sh
# For suse
source ./setup_suse.sh
# For macos
source ./setup_darwin.sh
```
